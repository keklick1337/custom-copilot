Global Settings

Enables VS Code's anonymous chat access (chat.allowAnonymousAccess) so you can use the Chat view with your own providers while signed out of GitHub. Requires reloading the window to take effect.
By default VS Code clears the open chat panel on a fresh start, so your conversations disappear after restarting (especially when signed out of GitHub). Enabling this sets chat.restoreLastPanelSession to true so the last chat session is restored from disk on startup. Chat history files are always written to disk; this only controls whether they are re-opened.
VS Code reports the selected model id in its core chat telemetry (interactiveSessionProviderInvoked), which can reveal which third-party API providers and models you use. Enabling this sets telemetry.telemetryLevel to off. Recommended for anonymity.
